## 目录树

```
~
├── my-ublock-static-filters   # chromium extension ublock 配置
├── .zshrc                     # zsh 配置文件
├── .xinitrc                   # x 窗口启动文件
├── .vimrc                     # vim 配置文件
├── git-push.sh                # git push script
├── keyboard-backup.pro        # 键盘配置文件             
│
├── .config/                   
│   ├── i3/
│   │   ├── config             # i3 配置文件
│   │   ├── wallpaper/         # 壁纸目录
│	│   │   ├── cmd.sh     # 执行壁纸脚本
│	│   │   ├── pictures/  # 壁纸图片文件夹
│   ├── fcitx/                 # 输入法配置目录
│   │   ├── config             # fcitx 配置文件
│   │   ├── skin/              # fcitx 皮肤
│   │   ├── pinyin/            # fcitx 词典

```
